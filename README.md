[Website](hhttps://mrbanano.github.io/Curriculum-vitae-react/)


 **Curriculum-vitae** is a website developed with `React.js`and `Styled components`,this project is part of the content of the facebook circle developer community, it is a complete project from installation and configuration to deployment in a production environment.

This project aims to show a developer's resume, it is consuming an api stored in firebase, using the most current standards within react such as functional components and custom hooks to query the information through fetch.

The project was stylized under the rules of style component, git was used for version control and travis was used as an intermediary to compile the application and be able to inject it into the host that is github page.

A grip system is used to be able to place the elements in the DOM and give it the characteristic of being adaptable on any screen size
